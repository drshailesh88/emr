# Keyboard Shortcuts Guide

DocAssist EMR includes comprehensive keyboard shortcuts to help you work faster and more efficiently.

## Quick Access

Press **F1** or **Ctrl+/** at any time to view all available keyboard shortcuts in an interactive overlay.

## Complete Shortcuts Reference

### General Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+N** | New Patient | Open the new patient dialog |
| **Ctrl+S** | Save | Save current form/visit |
| **Ctrl+M** | Toggle Voice | Start/stop voice recording |
| **Ctrl+Enter** | Submit | Submit/save and continue |
| **Escape** | Cancel | Close dialog or cancel current action |

### Prescription & Clinical

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+G** | Generate Rx | Generate prescription using AI |
| **Ctrl+P** | Print PDF | Generate and print prescription PDF |

### Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+F** | Focus Search | Jump to patient search box |
| **Ctrl+1** | Prescription Tab | Switch to Prescription view |
| **Ctrl+2** | Timeline Tab | Switch to Timeline view |
| **Ctrl+3** | Growth Tab | Switch to Growth Dashboard |
| **Ctrl+4** | Settings Tab | Switch to Settings view |
| **Alt+←** | Previous Patient | Navigate to previous patient in list |
| **Alt+→** | Next Patient | Navigate to next patient in list |

### System

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl+B** | Backup | Create manual backup |
| **Ctrl+,** | Settings | Open settings dialog |
| **F1** | Help | Show keyboard shortcuts help |
| **F5** | Refresh | Refresh patient list |

## Platform Differences

### macOS
On macOS, **Ctrl** is automatically mapped to **Cmd** (⌘) for most shortcuts, following standard Mac conventions.

### Web Version
Some shortcuts may conflict with browser shortcuts. In such cases, the browser shortcut will take precedence.

## Tips for Power Users

1. **Learn gradually**: Start with the most common shortcuts (Ctrl+N, Ctrl+S, Ctrl+F) and add more as you become comfortable.

2. **Tooltips**: Hover over buttons to see their associated keyboard shortcuts in the tooltip.

3. **Context-aware**: Some shortcuts only work when the relevant panel or form is active.

4. **Voice input**: Use Ctrl+M to quickly toggle voice recording without reaching for the mouse.

5. **Navigation**: Use Alt+← and Alt+→ to quickly move between patients without clicking.

## Troubleshooting

### Shortcut not working?

1. **Check focus**: Some shortcuts require focus on a specific panel
2. **Input fields**: When typing in a text field, only global shortcuts (F1, Escape) work
3. **Dialogs open**: Close any open dialogs first
4. **Browser conflicts**: Try the alternative shortcut (e.g., F1 instead of Ctrl+/)

### Accidental triggers?

- Use **Escape** to cancel most actions
- Undo is available for most data changes
- Backups are created automatically to protect your data
